# ls命令

语法格式：

`ls [-a -l -h] [linux路径]`



- -a: all的意思，列出全部文件，包含隐藏的文件和文件夹

- -l：以列表的形式展示内容，并展示更多的信息

- -h: 以易于阅读的形式列出文件的大小，如：K、M、G。必须搭配`-l`一起使用





语法中的选项可以组合使用，比如`-a`和`-l`可以组合使用。

写法：

- ls -l -a

- ls -al

- ls -la

上述三种写法，效果是相同的



## 举例：

易于阅读的形式列出/home/test目录下的全部内容

```shell
ls -alh /home/test
```
